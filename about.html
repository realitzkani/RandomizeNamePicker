About

The random name picker built for real sessions.

Random Name Picker is a free tool that makes choosing a name fair, fun, and live. Whether you're a teacher picking students, a streamer running a giveaway, or hosting a game night — this tool has everything you need.

It was built to go beyond the basics: real accounts, live multiplayer sessions, leaderboards, streaks, music, and more — all in one place, completely free.

Features

🎲
Random Picker
Slot-machine style roll with customizable speed and duration.
📡
Live Sessions
Share a code and let viewers join in real time.
🏆
Leaderboard
Track how many times each name has been picked.
🔥
Streaks
Highlights when the same name gets picked multiple times.
🎵
Music Player
Play Spotify, YouTube, or Apple Music while you pick.
💬
Live Chat
Viewers can chat in real time during the session.
🗂
Groups
Organize names into teams and pick from specific groups.
👑
Roles
Owner, Admin, and Viewer roles with custom permissions.

Who Is It For?

Teachers — Call on students fairly. Let them submit their own names. Track who's been picked.

Streamers & Content Creators — Run live giveaways with your audience watching in real time.

Game Night Hosts — Pick teams, assign roles, keep things random and fun.

Anyone — Any time you need a fair, random choice.
